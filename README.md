# customer_behavior_analysis
I developed a data analytics project showcasing customer behavior analysis using python,sql and power Bi.
📊 Customer Shopping Behavior – Data Analytics Project

🔎 Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from raw data processing to dashboard development and business insight generation.
The objective of this project is to analyze customer shopping behavior, identify key trends, and present actionable insights using Python, SQL, and Power BI.
This project highlights skills in:
Data Cleaning
Exploratory Data Analysis (EDA)
SQL Querying
Dashboard Development
Business Reporting

📁 Dataset
The dataset contains customer shopping and transaction-related information, including:
Customer details
Purchase amounts
Product categories
Subscription status
Review ratings
Age group segmentation
Shipping type
The data was initially loaded and explored using Python before further analysis.

Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard & Data Visualization)

Jupyter Notebook
Microsoft Excel (if applicable)
